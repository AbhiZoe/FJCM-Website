## ğŸ¨ Customization

### Update Church Information
Edit the following in `index.html`:

- **Church Name**: Change "FJCM" and "Faith Journey Church Ministry"
- **Service Times**: Update the service times in the Services section
- **Events**: Modify upcoming events in the Events section
- **Contact Info**: Update address, phone, email, and office hours in the Contact section

### Colors
Edit `styles.css` to change colors:
- Primary Color: `#667eea` (purple/blue)
- Dark Blue: `#1e3c72` and `#2a5298`
- Gold Accent: `#ffd700`

### Sections
The website includes the following sections:
1. **Home/Hero**: Welcome message with CTA button
2. **About Us**: Church mission and vision
3. **Services**: Service times and schedules
4. **Events**: Upcoming church events
5. **Contact**: Contact form and information
6. **Footer**: Copyright and social links
